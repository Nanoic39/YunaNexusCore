# 【Server】YunaNexusCore - 系统核心模块后端

## 选用技术栈
本系统选用技术：
- 应用开发：SpringBoot
- 微服务治理：SpringCloudAlibaba
- API 网关：Gateway
- 服务注册配置：Nacos
- 数据存储：MySQL+MyBatisPlus+Redis
- 熔断限流：Sentinel
- 消息队列：RocketMQ
- 分布式事务：Seata
- 链路追踪：SkyWalking+Elasticsearch+Prometheus+Node Exporter+AlertManager
- 任务调度：Spring Scheduler
- 邮箱告警：SpringBootStarterMail
